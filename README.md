# 📱 Portfolio App

## 🚀 Overview

The Portfolio App is a personal Android application that showcases profile details, skills, projects, and contact information. It acts as a digital resume with a clean and modern UI.

---

## ✨ Features

* 👤 Display personal profile (Name, Bio, Image)
* 💻 Showcase skills and technologies
* 📂 Display projects
* 📞 Contact section
* 🎨 Clean Material UI design

---

## 🛠 Tech Stack

* Java
* XML
* Android Studio
* Material Design Components
